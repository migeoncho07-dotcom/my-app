# 디자인 핸드오프 — 아이랑 (애플 톤 v2)

> 이 문서는 **디자인만** 다룹니다. Claude Code가 아래 시안을 보고 **UI를 그대로 구현**할 수 있게 작성했습니다.
> 기능·기술 스택·파싱 방식은 이 문서에서 정하지 않습니다 → 같은 폴더 **`기능요건서.md`** 참고.

## 보는 법
- 시안 파일: **`아이랑 시안 v2 애플.dc.html`** (+ `support.js`) — 브라우저로 바로 열림
- 한 페이지에 모든 화면이 가로로 나열돼 있습니다. 좌→우로 스크롤.
- 폰 프레임은 **393 × 768** 기준, 모바일 디자인 폭은 **375px**.

---

## 1. 디자인 방향 — "애플 톤"
iOS 시스템 UI 느낌. 쿨한 시스템 그레이 배경 + 흰 카드 + 헤어라인 보더 + 부드러운 중립 그림자.

- **라지 타이틀** — 홈 상단에 27px 굵은 좌측 정렬 타이틀
- **검색 필드** — 라지 타이틀 아래 반투명 그레이 캡슐 (`rgba(118,118,128,.12)`)
- **세그먼티드 컨트롤** — 카테고리 필터(전체/카페/호텔/야외). 선택 칸만 흰색 + 미세 그림자
- **글래스 탭바** — 반투명 흰색 + `backdrop-filter: blur` + 0.5px 헤어라인 상단선, 카드 위에 떠 있음
- **인셋 그룹 리스트** — 카드 상세의 정보 행은 둥근 사각형 그룹 + 행 사이 헤어라인 구분선

## 3. 타이포 — Pretendard
한글에서 SF Pro에 가장 가까운 폰트. CDN으로 로드되어 있음.
- 라지 타이틀 27 / 화면 타이틀 21~25 / 카드 제목 16~18 / 본문 13~14.5 / 보조 11.5~12.5
- weight: 제목 800, 강조 700, 본문 500, 보조 400~500. letter-spacing 제목 약 `-.03em`

## 4. 간격 · 모서리 · 그림자
- 모서리: 폰 베젤 50 / 화면 42 / 카드 18~22 / 필드·버튼 14~16 / 칩 7~11
- 그림자: 중립(`rgba(0,0,0,…)`). 카드 `0 6px 20px -14px rgba(0,0,0,.45)`, 버튼은 포인트색 그림자 옅게
- 폰 프레임 그림자만 깊게, 내부 요소는 옅게

## 5. 화면 목록
**핵심 5** — 로그인 · 홈/리스트 · 추가(미리보기) · 카드 상세 · 지도
**가입 플로우** — 로그인 / ① 이메일가입 → ② 초대코드 → ③ 가입완료 → ④ 프로필 → ⑤ 관심·동네 → ⑥ 환영 / 초대장 보내기
**정보 추가 플로우** — ① 붙여넣기(텍스트·사진·링크) → ② 자동 정리 → ③ 골라 담기 → ④ 확인·편집 → ⑤ 카카오 주소 → ⑥ 저장 완료

## 6. 컴포넌트
- **장소 카드** — 92px 썸네일 + 카테고리 배지 + 제목 + 지역·연령 + 등록자
- **세그먼티드 컨트롤 / 칩** — 필터. 지도에서는 핀 위 플로팅 펄(Apple Maps식)
- **글래스 탭바** — 홈/지도/＋/멤버/나. 중앙 ＋는 포인트색 라운드 버튼이 살짝 떠 있음
- **버튼** — Primary(포인트색 채움) / Secondary(흰 배경 + 헤어라인)
- **입력 필드** — 흰 배경 + 헤어라인, 포커스 시 포인트색 1.5px 보더
- **인셋 정보 행** — 아이콘 + 라벨/값, 행 사이 헤어라인

## 7. 구현 시 주의 (플레이스홀더)
- **지도** 배경/핀, **카드 이미지**, **카카오 검색 결과**는 시안에서 더미입니다 → 실제 지도 SDK·이미지·API 결과로 교체
- 아이콘은 인라인 SVG → 아이콘 라이브러리(SF Symbols 류)로 대체 가능
- 9:41 상태바·하단 인디케이터는 OS 영역, 실제 구현에서 제외
